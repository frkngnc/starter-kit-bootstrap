
# Gulp + Pug + Sass Bootstrap 4 Starter-Kit

## Usage

### 1. Clone repo
```
git clone https://github.com/frkngnc/starter-kit-bs4.git
```

### 2. Go inside cloned repo
```
cd starter-kit-bs4
```

### 3. Install all dependencies 
```
npm install
```

### 4. Run default gulp task
```
gulp dev
```

## Build 
```
gulp build
```
